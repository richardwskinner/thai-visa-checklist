# News Tagging Guide

Use these categories for `lib/data/visa-news.ts` and the news page pills/filters.

## Categories

### `Policy` 🟦
Long-term rule changes.

Use when:
- Visa-free stay length changes
- New visa types
- Eligibility rule adjustments
- Cabinet approvals affecting immigration

Rule of thumb: structural changes to the system.

### `Border` 🟩
Entry and enforcement.

Use when:
- Visa-run scrutiny
- Entry refusals
- Airport screening updates
- Arrival procedures

Rule of thumb: anything affecting how people enter Thailand.

### `Reporting` 🟨
Compliance while inside Thailand.

Use when:
- 90-day reporting
- TM.30 updates
- TDAC changes
- Extension process updates
- Overstay procedures

Rule of thumb: managing stay requirements after arrival.

### `Fees` 🟧
Money and financial thresholds.

Use when:
- Bank balance requirements
- Income thresholds
- Overstay fines
- Application fees
- Passenger charges

Rule of thumb: anything affecting cost or financial proof.

### `Official` 🟪
Direct government announcements.

Use when:
- Immigration Bureau notices
- MFA announcements
- Royal Gazette publications
- Formal government releases

Rule of thumb: use when the authority source is the key angle.

### `Life` 🟫
General Thailand updates (light but relevant).

Use when:
- Public holidays affecting travel
- Tourism promotions
- Transport improvements
- Driving licence system changes
- Things expats should know

Examples:
- "5-Day New Year Holiday Approved"
- "Online Driving Licence Renewals Launch"

Rule of thumb: not strictly visa rules, but useful for foreigners living in Thailand.

## Tagging Notes

- Choose the single strongest angle for the headline.
- Prefer `Official` only when the announcement source itself is the main point.
- If an item is mainly about cost/charges, prefer `Fees` even if it is travel-related.
- If an item affects entry checks or refusals, prefer `Border`.
